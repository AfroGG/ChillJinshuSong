# 寒蝉锦书宋 
寒蝉锦书宋(ChillJinshuSong)基于思源宋体(SourceHanSerif)修改，添加圆角，更加温润的宋体，排版细节优化，重构字重比例，字体粗细更加均衡，便于阅读。名为锦书体，既是锦上添花，又是前程似锦之意。
